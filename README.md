# Quiz Application

A fully functional quiz application built using **HTML**, **CSS**, and **JavaScript**. This project provides a dynamic and user-friendly interface for users to test their knowledge across multiple categories and track their performance.

---

## Features
- **User Authentication**: Login and registration screens for user identity.
- **Dynamic Quiz Flow**: Timer-based questions with options to skip or revisit unanswered ones.
- **Accurate Scoring**: Tracks correct answers while ignoring skipped and incorrect responses.
- **Interactive UI**: Highlights correct, incorrect, and unanswered questions with color coding.
- **Responsive Design**: Optimized for various screen sizes, ensuring seamless use on mobile and desktop devices.

---

## Technologies Used
- **HTML**: Structuring the app's layout and content.
- **CSS**: Styling for a modern and intuitive user experience.
- **JavaScript**: Managing quiz logic, question navigation, and scoring.
